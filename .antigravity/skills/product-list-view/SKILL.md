# 產品列表視圖技能 (Product List View Skill)

此技能定義了產品列表（首頁）的視覺風格與行為。

## 1. UI 結構
-   **佈局**: 採用標準表格佈局 (Standard Table Layout)，而非卡片式設計。
-   **欄位**:
    1.  **客戶 (Client)**: 文字，靠左對齊。
    2.  **型號規格 (Model Spec)**: 文字，靠左對齊。
    3.  **上月庫存 (Last Month Inv)**: 數字，靠左對齊。
    4.  **動作 (Action)**:  按鈕 ("編輯"), 按鈕(查詢歷史)，置中或靠左對齊。
-   **樣式**:
    -   標題背景: 淺藍色 (`var(--secondary-color)`)。
    -   懸停效果: 滑鼠指向行時顯示淺色背景。
    -   邊框: 極簡的水平邊框。

## 2. 控制區域 (Controls)
-   **位置**: 列表上方。
-   **元素**:
    1.  **客戶篩選**: 下拉選單 (`<select>`)。預設值："所有客戶"。
    2.  **統計資訊**: 文字 "顯示 X 筆，共 Y 筆" (灰色字體)。
    3.  **新增按鈕**: 靠右對齊，使用藍色主色調。

## 3. 互動行為
-   **篩選**: 切換客戶下拉選單時，必須立即過濾表格行，無需重新載入頁面。
-   **新增**: 點擊 "新增產品" 時，開啟 `<dialog id="dialog-add">`。
-   **編輯**: 點擊行內的 "編輯" 按鈕，跳轉至詳細預測頁面。

## 4. 語言規範
-   嚴格遵守 `localization-rules` 技能。
-   **不顯示** 搜尋框 (已移除之功能)。
