# PSI 年度歷史視圖技能 (PSI Yearly History View Skill)

此技能定義了產品年度歷史資料頁面的顯示規範與互動邏輯。

## 1. UI 結構
-   **控制列 (Controls)**:
    -   **顯示年度**: 下拉選單 (`2024`, `2025`, `2026` 等)。預設為目前年度。
    -   **產品資訊**: 顯示目前檢視的「客戶」與「型號規格」。
    -   **返回按鈕**: 返回詳細預測頁面或產品列表。
-   **歷史網格 (History Table)**:
    -   **欄位**: `月份 (YYYY-MM)`, `預測到貨 (P)`, `預測銷售 (S)`, `庫存 (I)`, `最後更新時間`.
    -   **顯示**: 條列式顯示 1 月至 12 月的完整資料。
    -   **樣式**: 採用與首頁一致的專業表格風格。

## 2. 數據邏輯
-   **隨選載入 (On-Demand Loading)**: 進入此頁面時，根據選定的 `client`, `model_spec` 與 `year` 向 API 請求資料。
-   **年度切換**: 改變「顯示年度」下拉選單時，必須立即觸發 API 請求並更新表格內容。
-   **資料來源**: 對應 Google Sheets 中名為該年度（如 "2026"）的分頁。

## 3. 互動行為
-   此頁面主要為 **唯讀 (Read-only)**，用於查閱歷史軌跡。若要修改資料，應引導使用者回詳細預測頁面。

## 4. 語言規範
-   嚴格遵守 `localization-rules` 技能。
-   標題範例：`[客戶名稱] [型號規格] - 2026 年度歷史資料`。
